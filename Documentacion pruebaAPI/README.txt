README - Desafio Backend

Sobre el API
- El Api fue creado en el Editor Visual Studio Code actualizado a la fecha, con las extensiones para trabajar en C#.
- La conexion a la base de datos fue realizada en la clase Startup.cs donde se realizo los parametros y credenciales de la BD
- Se realizo la configuracion Swagger dentro del proyecto y se deplego perfectamente en el navegador.

Sobre la BD
- La BD fue realizada en el  motor de base de datos Microsoft SQL Server. En la misma se creo de manera sencilla la bd llamada 'HOGWARTS' y la tabla 'Brujas_Magos'

- Se deja los siguientes archivos que se detallaran a continuacion con el fin de facilitar su revision:

	- 'database.sql' es el script de la creacion de toda la base de datos que se uso en Microsoft SQL Server.
	- 'dataprueba.sql' es donde se encuentra el script de creacion de la tabla 'Brujas_Magos' las columnas y la insercion de los datos de prueba.
	- 'pruebaAPI.postman_collection.json' es la coleccion de Postman que detalla el API JSON que se uso para realizar las pruebas de Create, Read, Update, Delete, de los registros usados.
